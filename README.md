## Project Description
This project demonstrates a full CI/CD pipeline using:

- Docker
- Kubernetes
- GitHub Actions
- Argo CD

## Workflow
1. Code push triggers GitHub Actions
2. Docker image is built and pushed
3. Kubernetes deployment file is updated
4. Argo CD deploys automatically
